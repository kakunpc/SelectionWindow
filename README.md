# SelectionWindow

## Install
`Packages/manifest.json` add
```
"com.kakunpc.selectionwindow": "https://github.com/kakunpc/SelectionWindow.git"
```

or 
`Packages/manifest.json` add
```
{
  "scopedRegistries": [
    {
      "name": "kakunpc.com",
      "url": "https://upm.kakunpc.com",
      "scopes": [
        "com.kakunpc"
      ]
    }
  ]
}
```

and install to UnityPackageManager
